Documentation Page

This is a sample documentation page demonstrating the Athena DevDocs widget.

Press Cmd+K (Mac) or Ctrl+K (Windows/Linux) to open the search widget, or click the button in the bottom-right corner.

Getting Started

To install the SDK, run:

npm install @acme/sdk

Authentication

Pass your API key in the Authorization header:

curl -H "Authorization: Bearer YOUR_KEY" https://api.acme.com/v1/data
