## PART 3: Code Quality & Technical Debt

### Critical Issues (Fix Before Production)

#### 1. N+1 Query Pattern — `routes.py:208-218`, `routes.py:263-273`

```python
# CURRENT (11 queries for 10 chunks)
for chunk, score in chunks_with_scores:
    doc_result = await db.execute(select(Document).where(Document.id == chunk.document_id))
    doc = doc_result.scalar_one_or_none()
```

Fix with batch query or `selectinload`. This doubles latency on every `/query` call.

#### 2. Rate Limiter Is In-Memory Only — `middleware.py:135-184`

Breaks with multiple replicas. Move to Redis (already a dependency).

#### 3. Credential Hygiene

`.env` has real ZhipuAI and OpenRouter keys. While not committed to git (`.gitignore` is correct), rotate them as a practice. Consider using a secrets manager.

#### 4. Static Key Fallback — `middleware.py:75-78`

When no tenants exist in DB, the system allows unauthenticated requests. This is a security gap during initial deployment.
